My first version control commit!

Vc can be defined as an important part of every software development

The process of committing a project to github repository are:

  • open a folder,inside your folder open a file
  • Name your file, your repository name should be the same as your file name
  • move your file into your vsc and give it a name
  • then you start by pressing an exclamation mark then enter
  • then you git init
  • then your first commit
  • then your git status
  • then your git add
  • you can use the up and down arrow for your history
  • then you link your repository to your github
  • then push
